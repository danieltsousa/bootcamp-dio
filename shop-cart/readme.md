Este projeto aplica componentes e funções que seriam necessárias para a lógica de programação de um serviço de um carrinho de compras online  para qualquer loja virtual, incluindo as ações de adicionar, remover e excluir items nesse carrinho, que é uma espécie de lista onde cada produto possui informações necesárias para estarem nessa lista.
